### Deployment Modes

Webroamer OS is designed to be versatile:

1. **Web GUI**: A fully functional UI accessible directly via any modern standards-compliant browser.
2. **Containerized**: Native Docker mounting (e.g., file-manager-like).
3. **OS-Native**: Designed for future physical machine installation.
4. **Remote System**: Browser-based remote desktop access to other Webroamer system.
